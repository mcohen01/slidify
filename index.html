Linear Regression

Linear Regression is a great technique for initial exploration into the world of predictive modeling.

Small Datasets

Small datasets are much easier to work with and therefore more appropriate for the beginner. For instance, we can quickly run a linear regression on the mtcars dataset right in this presentation!

library(datasets)
data(mtcars)
fit <- lm(mpg ~ wt, data=mtcars)
summary(fit)
